02. Ayuda y Manuales

Nadie memoriza todos los comandos. Así pedimos ayuda:

man ls Manual: Muestra el manual completo de un comando. (Q para salir).
help cd Ayuda rápida: Información breve de comandos internos.
clear Limpia la pantalla para trabajar más cómodos.

03. Navegación (pwd, cd)

pwdPrint Working Directory. Ruta actual.
cd DescargasEntrar a una carpeta.
cd ..Salir a la carpeta anterior.
cd ~Ir a mi carpeta personal (HOME).
Símbolo ~ (Virgulilla): AltGr + 4  o  Alt + 126

04. Listar Archivos (ls)

ls

Listado simple.

ls -la

Todo y Largo: Detalles y archivos ocultos.

dir

Alternativa que viene de Windows.

ls -t

Ordena por fecha (más recientes primero).

05. Crear y Borrar (Peligro)

mkdir carpetaCrea un directorio nuevo.
rm archivo.txtBorra un archivo (File).
rmdir carpetaBorra carpeta SOLO si está vacía.
rm -r carpetaRecursivo: Borra la carpeta y TODO su contenido.

06. Copiar (cp) y Mover (mv)

cp archivo destinoCopia un archivo.
cp -r dir1 dir2Copia una carpeta completa.
mv archivo destinoMueve un archivo.
mv viejo.txt nuevo.txtRenombrar: Así cambiamos nombres en Linux.

07. Leer Archivos

cat notas.txt

Muestra TODO el contenido de golpe.

less notas.txt

Visor para leer poco a poco (Q para salir).

head -n 5 log.txt

Lee solo las primeras 5 líneas.

tail -n 5 log.txt

Lee solo las últimas 5 líneas.

08. ¿Cómo creo un archivo?

1. Archivo Vacío

touch notas.txt

2. Archivo con texto rápido

echo "Hola mundo" > saludo.txt

3. Escribir en vivo

cat > lista.txt

Escribes tus líneas en la consola... y para guardar y salir presionas:

CTRL + D

09. Buscar Archivos (find)

Busca archivos y directorios en el sistema.

find . -name "tarea.txt" Busca el archivo exactamente con ese nombre.
find . -iname "TaRea.txt" Ignora Mayúsculas: Lo encuentra sin importar cómo esté escrito.
find . -name "*.pdf" Usa comodín: Busca todos los archivos que terminen en .pdf

10. Buscar Texto (grep)

Busca palabras DENTRO de los archivos.

grep "error" log.txtMuestra líneas con la palabra "error".
grep -i "Admin" log.txtIgnore: No distingue mayúsculas.
grep -c "root" log.txtCount: Muestra CUÁNTAS VECES aparece.
grep -v "root" log.txtInvert: Muestra todo lo que NO tenga la palabra root.

11. Filtros (wc, sort)

Contador (wc)

wc -l archivo.txt

Cuenta cuántas líneas (-l) tiene el archivo.

Ordenamiento (sort)

sort archivo.txt

Ordena el contenido alfabéticamente.

wc -w archivo.txt

Cuenta palabras (-w).

sort -r archivo.txt

Ordena de forma inversa (Z-A).

12. Procesos (top, ps, kill)

Un "proceso" es un programa ejecutándose.

top Monitor en tiempo real. (Presiona Q para salir).
ps -aux Muestra todos los procesos actuales y su número PID (ID).
kill 1234 Cierra a la fuerza un programa colgado usando su PID (ej: 1234).

13. Permisos (Modo Letras)

Fórmula: [Quién] [Signo] [Permiso]

u (Usuario)
g (Grupo)
o (Otros)
+ / -
r (Leer)
w (Escribir)
x (Ejecutar)
chmod u+x script.sh

Significa: "Al Usuario, +sumarle, eXecución".

14. Permisos (Modo Números)

Lectura (r)= 4
Escritura (w)= 2
Ejecución (x)= 1
chmod 755 archivo

Dueño (4+2+1=7) | Grupo (4+1=5) | Otros (4+1=5)

15. Cambiar el Dueño (chown)

Transfiere la propiedad de un archivo o carpeta a otro usuario.

chown nuevo archivo.txt Pasa la propiedad al usuario "nuevo".
chown -R root:admin /dir Recursivo: Le da la carpeta y su contenido al usuario 'root' y al grupo 'admin'.

16. Los 3 Flujos (Concepto)

En Linux "todo es un archivo", incluso los canales de comunicación:

Standard Input

stdin (0)

La entrada de datos (tu teclado).

Standard Output

stdout (1)

La salida exitosa (tu pantalla).

Standard Error

stderr (2)

Por donde salen los mensajes de error.

17. Redirecciones (>, >>, 2>)

Desviar la salida de la pantalla hacia un archivo.

echo "Hola" > saludo.txt Sobrescribir (>): Crea o sobrescribe el archivo.
echo "Mundo" >> saludo.txt Añadir (>>): Agrega texto al final sin borrar nada.
ls /falsa 2> error.log Error (2>): Guarda el mensaje de error en un archivo.

18. Tuberías (Pipes |)

El pipe (|) toma el resultado de un comando y lo usa como entrada del siguiente.

ls -la | grep "Feb" | wc -l
1. ls
Lista archivos
2. grep
Filtra "Febrero"
3. wc
Cuenta líneas
