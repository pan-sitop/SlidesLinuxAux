02. Navegación (Moverse)

pwd ¿Dónde estoy parado? (Ruta).
cd Descargas Entrar a una carpeta.
cd .. Salir a la carpeta anterior (Atrás).
cd ~ Ir a mi carpeta personal (HOME).
¿No te sale el símbolo ~?

Intenta con estos atajos:

AltGr + 4  o  Alt + 126

03. Listar Archivos

ls

Listado simple (solo nombres).

ls -la

Combo: Muestra TODO (ocultos) y DETALLES (permisos).

dir

Para Windows: Funciona igual que ls en la mayoría de Linux.

ls -t

Tiempo: Ordena por fecha de modificación.

04. Comodines (Selección Múltiple)

Para seleccionar muchos archivos a la vez.

* El Asterisco: Significa "CUALQUIER COSA".
ls *.jpg (Muestra todos los archivos JPG).
? Interrogación: Significa "UN CARACTER".
ls foto?.png (Busca foto1, fotoA, fotoX).

05. Crear Directorios (Carpetas)

mkdir carpeta

Crea una carpeta nueva vacía.

mkdir -p a/b/c

Cascada: Crea una carpeta dentro de otra automáticamente.

mkdir "Mi Carpeta"

Usa comillas si el nombre tiene espacios.

tree

Muestra tus carpetas en forma de árbol (si está instalado).

06. El Arte de Borrar (¡Cuidado!)

En Linux NO existe la Papelera de Reciclaje.
rm archivo.txt Borra un archivo (File).
rmdir carpeta Borra carpeta SOLO si está vacía.
rm -r carpeta Recursivo: Borra la carpeta y TODO lo que tiene adentro.
rm -i archivo Interactivo: Te pregunta "¿Seguro?" antes de borrar.

07. Copiar (cp) y Mover (mv)

cp archivo destino Copia un archivo.
cp -r carpeta1 carpeta2 Recursivo: Obligatorio para copiar carpetas completas.
mv archivo destino Mueve un archivo de lugar.
mv viejo.txt nuevo.txt Renombrar: Linux no tiene comando "rename", usamos mv.

08. Leer y Unir (cat)

cat archivo

Muestra todo el contenido rápido.

less archivo

Permite navegar (arriba/abajo). Salir con 'Q'.

head -n 5

Primeras 5 líneas.

tail -n 5

Últimas 5 líneas (Logs).

09. ¿Cómo creo un archivo?

1. Archivo Vacío

touch notas.txt

2. Archivo con texto rápido

echo "Hola mundo" > saludo.txt

3. Escribir en vivo

cat > lista.txt

Escribes tus líneas... y al final presionas:

CTRL + D

10. Buscar (Find y Grep)

find

Busca ARCHIVOS.

find . -name "*.pdf" Busca todos los PDFs aquí.

grep

Busca TEXTO dentro.

grep "root" /etc/passwd Busca la palabra "root".

11. Permisos (Modo Letras)

Fórmula: [Quién] [Signo] [Permiso]

u (Usuario)
g (Grupo)
o (Otros)
+ / -
r (Leer)
w (Escribir)
x (Ejecutar)
chmod u+x script.sh

Significa: "Al Usuario, +sumarle, eXecución".

12. Permisos (Modo Números)

Lectura (r)= 4
Escritura (w)= 2
Ejecución (x)= 1
chmod 755 archivo

Dueño (7) | Grupo (5) | Otros (5)

13. Tuberías (Pipes |)

Pasar la información de un comando a otro.

cat lista.txt | sort

Lee la lista -> La ordena alfabéticamente.

ls -la | less

Lista directorio -> Permite hacer scroll.

14. Scripting Básico

  1. Crear archivo .sh
  2. Cabecera: #!/bin/bash
  3. Permisos: chmod +x
  4. Ejecutar: ./script.sh
#!/bin/bash
echo "Hola, ¿tu nombre?"
read nombre
echo "Creando carpeta..."
mkdir "$nombre"
echo "¡Listo!"
